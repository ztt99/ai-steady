函数调用时：

Function Execution Context Creation
    1. 创建 function scope
    2. 绑定 parameters
    3. 初始化 arguments
    4. var hoisting

```js
function foo(a) {
  console.log(a); // undefined
  var a = 5;
}
foo(10);
```
真实行为：

parameter binding
a = 10

var hoisting
var a (ignored because parameter exists)

execution
console.log(a) -> 10
a = 5


parameter 和 var 同名
parameter 优先
